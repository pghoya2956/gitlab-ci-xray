# AP-009: needs 순환 의존

| 항목 | 값 |
|------|-----|
| 심각도 | error |
| 카테고리 | 구조 |
| 자동 수정 | — |
| 도입 | v0.1.0 |

## 설명

`needs` 그래프에 순환이 있으면 GitLab CI 엔진이 파이프라인 생성을 거부합니다.

## 근거

DAG(Directed Acyclic Graph)는 정의상 순환이 없어야 합니다. A → B → A 같은 순환이 있으면 어떤 job도 시작할 수 없습니다.

## 탐지

DFS(깊이 우선 탐색)로 `needs` 그래프의 순환을 탐지합니다.

## 예시

```yaml
# Bad — 순환: a → b → a
a:
  needs: [b]
  script: echo a

b:
  needs: [a]
  script: echo b
```
