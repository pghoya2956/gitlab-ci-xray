# AP-012: rules 빈 배열

| 항목 | 값 |
|------|-----|
| 심각도 | error |
| 카테고리 | 구조 |
| 자동 수정 | — |
| 도입 | v0.1.0 |

## 설명

`rules: []`는 어떤 조건에서도 job이 실행되지 않음을 의미합니다.

## 근거

`rules`가 빈 배열이면 매칭되는 규칙이 없어 job이 항상 `when: never`로 평가됩니다. 의도적이라면 코드의 명확성을 위해 job을 제거하거나 주석 처리하는 것이 좋습니다.
