# AP-005: script/trigger 누락

| 항목 | 값 |
|------|-----|
| 심각도 | error |
| 카테고리 | 구조 |
| 자동 수정 | — |
| 도입 | v0.1.0 |

## 설명

job에 `script` 또는 `trigger`가 없으면 GitLab CI 엔진이 파이프라인 생성을 거부합니다.

## 근거

GitLab CI에서 모든 job은 `script` 또는 `trigger` 중 하나를 반드시 가져야 합니다. 이는 구조적 오류입니다.

> Each job must have a `script` keyword or a `trigger` keyword.
> — [GitLab CI/CD YAML reference](https://docs.gitlab.com/ee/ci/yaml/#script)

## 참고 사항

`include`가 미해석된 경우, extends를 통해 script를 상속받는 job에서 오탐이 발생할 수 있습니다. 이 경우 severity가 `info`로 자동 하향됩니다.
