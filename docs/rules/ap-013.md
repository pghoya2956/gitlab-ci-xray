# AP-013: image 미설정

| 항목 | 값 |
|------|-----|
| 심각도 | warning |
| 카테고리 | 재현성 |
| 자동 수정 | — |
| 도입 | v0.1.0 |

## 설명

`image`가 설정되지 않고 `default.image`도 없으면 러너의 기본 이미지에 의존합니다. 러너 설정에 따라 빌드 결과가 달라질 수 있습니다.

## 근거

재현 가능한 빌드를 위해 이미지를 명시적으로 지정해야 합니다. 러너 관리자가 기본 이미지를 변경하면 파이프라인이 갑자기 실패할 수 있습니다.

## 예외

- `trigger` job은 이미지가 필요 없음 (downstream 파이프라인 트리거만 수행)
- shell executor를 사용하는 러너는 이미지 개념이 없음 (이 도구에서는 구분 불가)
