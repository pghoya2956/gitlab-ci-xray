# AP-014: 정의되지 않은 stage

| 항목 | 값 |
|------|-----|
| 심각도 | error |
| 카테고리 | 구조 |
| 자동 수정 | — |
| 도입 | v0.1.0 |

## 설명

`stages`에 정의되지 않은 stage를 job이 참조하면 GitLab CI 엔진이 경고하거나 파이프라인이 예상과 다르게 동작합니다.

## 근거

GitLab CI는 정의되지 않은 stage를 사용하는 job을 `test` stage로 처리하거나 에러를 발생시킬 수 있습니다. 명시적으로 `stages`에 정의하는 것이 안전합니다.
